# GridOne Visual Assets

This directory holds source and export assets used across the repository, demo, and Devpost submission.

## Folders

- `ui/` — wireframes, screen flows, and final product captures.
- `diagrams/` — architecture, trust boundary, task lifecycle, and flywheel diagrams.
- `logos/` — editable logo sources, monochrome variants, and export files.
- `presentation/` — title cards, pitch slides, thumbnails, captions, and video assets.

## Conventions

- Keep editable sources and exported files together when licensing allows.
- Prefer SVG for diagrams and marks; PNG for screenshots and platform uploads.
- Use lowercase kebab-case names, for example `builder-dashboard-dark.png`.
- Add the source, creator, license, and date to `ASSET_LOG.md` before submission.
- Never add personal information, API keys, private screenshots, or unlicensed third-party assets.
